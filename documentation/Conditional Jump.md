Conditional jumps set the pcwrite microinstruction on if the flag is a desired value.

| Flag              | Selector value |
| ----------------- | -------------- |
| Carry             | `1`            |
| Zero              | `2`            |
| Negative          | `3`            |
| More Than         | `4`            |
| Equal             | `5`            |
| Less Than         | `6`            |
| Interrupt Disable | `7`            |
